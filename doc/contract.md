## Contract Information

**Request URL**
```
GET /contract/:contract
```

```

## Call Contract
Returns RPC `callcontract` result.

**Request URL**
```
GET /contract/:contract/call
```

**Request Parameters**
<table>
    <thead>
        <tr>
            <th>Name</th>
            <th>Type</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>
              <code>data</code>
            </td>
            <td>
              String
            </td>
            <td>Hexadecimal data to send to contract</td>
        </tr>
        <tr>
            <td>
              <code>sender</code>
            </td>
            <td>
              String (optional)
            </td>
            <td>Base58 P2PKH or 20-byte hexadecimal sender address</td>
        </tr>
    </tbody>
</table>

**Request**
```
GET /contract/6b8bf98ff497c064e8f0bde13e0c4f5ed5bf8ce7/call?data=313ce567
